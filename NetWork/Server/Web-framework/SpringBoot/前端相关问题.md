## npm一直停在"checking installable status"或者下载卡住的问题

* 原因：旧的npm缓存与项目冲突和资源下载不畅

解决方法：

步骤一

```cmd
(sudo) npm cache clean
```

步骤二

```cmd
npm config set registry https://registry.npm.taobao.org
```
